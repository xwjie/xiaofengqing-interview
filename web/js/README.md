# 源代码说明

> 使用mocha框架测试，需要先全局安装\
> npm install --global mocha

重点：
- 掌握递归写法
- 掌握基本的面向对象思维
- 使用数组的方法（push，pop，shift，unshift）实现堆栈等基本数据
  - shift：弹出并返回数组第一个元素
  - unshift：在数组第一个元素前插入元素
  - push：在数组末尾添加元素
  - pop：删除并返回数组末尾元素
