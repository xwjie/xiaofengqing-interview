defer async DOMContentLoaded

  • defer会影响DOMContentLoaded事件时间。
  • DOMContentLoadedBeforeDefer
  • DOMContentLoadedAfterDefer
  • async 不会影响DOMContentLoaded事件时间。
  • DOMContentLoadedBeforeAsync
  • DOMContentLoadedAfterAsync
  • DOMContentLoaded Multi Async, 之间没有顺序
  • DOMContentLoaded Multi Defer，多个之间按顺序执行
